## Application de gestion (scm-app)

Application permettant de géres les paramètres de votre établissement pour l'année scolaire, et de générer vos base de données pour vos utilisateurs.
Découvrez plus d'informations sur le site internet.

![gestion](https://github.com/user-attachments/assets/ba13d8c7-a6fd-4a12-ba2c-401ed148d4ce)

Plus d'informations & téléchargements : [Scm-app](https://schoolmore.fr/download/scmapp)

> [!NOTE]
> Application disponible sur le site officiel


## Application serveur (scm-serv)

Application permettant de mettre à disposition de vos utilisateurs la base de données éditée dans le logiciel de gestion. Cela permet également aux utilisateurs de l'application mobile de visualiser leurs données

![server](https://github.com/user-attachments/assets/82fe9d9a-ee83-4f2a-95b9-e77a463f7d73)


> [!WARNING]
> Cette application n'est pas disponible pour le moment


Plus d'informations & téléchargements : [Scm-serv](https://schoolmore.fr/download/scmserv)

## Application mobile (scm-mobile)

Application disponible sous Android (pour le moment) permettant aux utilisateurs de visualiser les données, évitant de passer systématiquement par le site web de l'établissement.

![mobile](https://github.com/user-attachments/assets/5ddd613b-53e2-4d1c-8331-a6df88ca3095)

> [!WARNING]
> Cette application n'est pas disponible pour le moment


Plus d'informations & téléchargements : [Scm-mobile](https://schoolmore.fr/download/scmmobile)
